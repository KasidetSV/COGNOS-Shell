### 2. The JoT-Auditor (Forensic RCA)
An automated diagnostic protocol that halts execution during high-complexity or divergent tasks. It forces a Root Cause Analysis (RCA) trace, ensuring that the AI agent does not silently fail or hallucinate code fixes.

* **T**rigger (Human-in-the-loop approval before compilation)

### 4. L2 → L1 Memory Distillation (Hermes Persistence)
COGNOS does not just execute; it learns. The system tracks successful problem resolutions and automatically promotes them from ephemeral session logs into permanent, hard-coded architectural laws.
